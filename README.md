# What it does?
By using chimu.moe api and checking for duplicated beatmaps every time it run, you can run this script anytime in background

# How to use (python >3.10)

```
git clone https://github.com/vuhz/Osu-Backup.git
cd Osu-backup
python osubackup.py
```
